# 专业负片相机 - 小米增强版

一款基于Web的实时负片相机应用，具有高级图像处理功能和小米设备协同增强特性。

## 功能特性

- **实时负片处理** - 毫秒级负片效果转换
- **高级参数调节** - 亮度、对比度、饱和度、HSL、颜色曲线等
- **小米设备协同** - 跨设备摄像头控制功能
- **多种滤镜效果** - 标准负片、黑白负片、高对比度、反相增强等
- **预设配置** - 复古、冷调、暖调、戏剧、清晰等多种预设
- **专业级控制** - 曲线编辑器、锐化、降噪等高级功能
- **响应式设计** - 适配各种设备屏幕尺寸

## 技术特点

- 纯前端实现，无需安装
- 基于WebRTC的摄像头访问
- Canvas实时图像处理
- 现代化UI设计，毛玻璃效果
- 高性能渲染，实时预览
- 支持拍照和图片下载

## 使用要求

- 现代浏览器（Chrome、Safari、Firefox、Edge）
- HTTPS协议或localhost环境
- 摄像头权限访问
- 支持WebGL和Canvas

## 安装与使用

1. 将 `negative-camera.html` 文件部署到支持HTTPS的Web服务器
2. 或者在本地环境中直接打开HTML文件（localhost）
3. 访问页面并点击"开始使用"按钮
4. 授予摄像头访问权限
5. 开始使用负片相机功能

## 功能说明

### 滤镜效果
- 标准负片：经典的负片效果
- 黑白负片：去色的负片效果
- 高对比度：增强对比度的负片
- 反相增强：强化反相效果
- 原图：恢复正常图像

### 参数调节
- 亮度：调整图像明暗程度
- 对比度：调整图像对比强度
- 饱和度：调整色彩饱和程度
- HSL：色相、明度、饱和度精细调节
- 颜色曲线：自定义颜色映射曲线
- 锐化：增强图像边缘清晰度
- 降噪：减少图像噪点

### 小米协同
- 设备扫描与连接
- 远程摄像头控制
- 参数同步功能
- 跨设备协同拍摄

## 操作说明

1. **启动应用**：点击"开始使用"按钮并授权摄像头访问
2. **切换摄像头**：点击切换摄像头按钮在前后摄像头间切换
3. **调整参数**：使用滑块调节各项图像参数
4. **选择滤镜**：点击滤镜选项应用不同负片效果
5. **拍照**：点击中央拍摄按钮进行拍照
6. **下载图片**：拍照后点击下载按钮保存图片

## 故障排除

- 如果无法访问摄像头，请检查浏览器权限设置
- 确保使用HTTPS协议或localhost环境
- 部分移动浏览器可能不支持所有功能
- 如遇性能问题，可尝试降低图像分辨率
